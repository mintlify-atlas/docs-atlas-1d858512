## Overview

The Dashboard screen serves as the main navigation hub after successful authentication. It provides quick access to key app features through large, icon-based menu cards.

**Route Path:** `/dashboard`

<Note>
  Users are redirected to this screen after successful login using `router.replace('/dashboard')` from the Login screen.
</Note>

## MenuCard Component

The `MenuCard` component creates large, touchable cards with icons:

```tsx
interface MenuCardProps {
    title: string;
    Icon: LucideIcon;
    onPress?: () => void;
}
```

### MenuCard Features
- **Large Icon:** 64px Lucide icons in brand color (#C91843)
- **Centered Layout:** Icon and text centered within the card
- **Interactive Feedback:** Active state changes background to gray-50
- **Bordered Design:** Subtle border with primary color at 20% opacity
- **Shadow Effect:** Soft shadow for depth
- **Rounded Corners:** `rounded-2xl` for modern appearance

### MenuCard Styling
```tsx
className="flex-1 border border-primary/20 bg-white rounded-2xl p-6 items-center shadow-sm active:bg-gray-50 m-2"
```

## Icon Library

The dashboard uses **Lucide React Native** for menu icons:

```tsx
import { FileText, User } from 'lucide-react-native';
```

<Note>
  Lucide icons are passed as components to MenuCard, which renders them with consistent size (64) and color (#C91843).
</Note>

## Layout Structure

```tsx
<ScreenLayout paddingHorizontal={4}>
    <Logo className="mt-8 mb-12" />

    <View className="space-y-4">
        <View className="flex-row">
            <MenuCard
                title="Configuración de cuenta"
                Icon={User}
                onPress={() => console.log('Configuración press')}
            />
        </View>

        <View className="flex-row">
            <MenuCard
                title="Documentos"
                Icon={FileText}
                onPress={() => console.log('Documentos press')}
            />
        </View>
    </View>
</ScreenLayout>
```

### Layout Details

- **ScreenLayout:** Uses `paddingHorizontal={4}` (16px) instead of default 6 (24px)
- **Logo Spacing:** `mt-8 mb-12` for prominent placement
- **Menu Container:** `space-y-4` for vertical spacing between cards
- **Card Rows:** Each `MenuCard` wrapped in `flex-row` View for potential multi-column layouts
