## Overview

The Register screen allows new users to create an account in DriveLoop Mobile. It features a comprehensive registration form with validation fields and terms acceptance.

**Route Path:** `/register`

## State Management

The screen uses React state to manage the terms and conditions acceptance:

```tsx
const [accepted, setAccepted] = useState(false);
```

This state controls the checkbox appearance and can be used for form validation before submission.

## Terms and Conditions

Interactive checkbox component for user consent:

```tsx
<TouchableOpacity
    className="flex-row items-center mt-4"
    onPress={() => setAccepted(!accepted)}
>
    <MaterialCommunityIcons
        name={accepted ? "checkbox-marked" : "checkbox-blank-outline"}
        size={24}
        color={accepted ? "#C91843" : "#6B7280"}
    />
    <Text className="ml-3 text-gray-500 text-sm font-roboto-light">
        Acepto los <Text className="text-blue-400">términos y condiciones</Text>
    </Text>
</TouchableOpacity>
```

<Note>
  The checkbox changes from `checkbox-blank-outline` to `checkbox-marked` when accepted, with color changing from gray to the primary brand color (#C91843).
</Note>

## Header Layout

The custom header combines navigation and title:

```tsx
<View className="flex-row items-center mt-4">
    <TouchableOpacity onPress={() => router.back()} className="p-2 -ml-2">
        <MaterialIcons name="arrow-back" size={24} color="#111111" />
    </TouchableOpacity>
    <View className="flex-1 items-center mr-8">
        <Text className="text-xl font-roboto-bold text-secondary">Registro</Text>
    </View>
</View>
```

This creates a balanced layout with the back button on the left and centered title.

## Styling

Key styling classes used:

- **Header:** `flex-row items-center mt-4` for horizontal layout
- **Title:** `text-xl font-roboto-bold text-secondary` for prominence
- **Logo:** `mt-6 mb-8` for reduced spacing compared to login
- **Form:** `space-y-2` for tighter field spacing
- **Button container:** `mt-10 mb-8` for spacing at bottom
- **Terms text:** `text-gray-500 text-sm font-roboto-light`
- **Terms link:** `text-blue-400` for emphasis

## Icon Libraries

The screen uses two icon libraries:

1. **MaterialIcons** - For form field icons (person, mail, lock)
2. **MaterialCommunityIcons** - For checkbox states

Both are imported from `@expo/vector-icons`.
