## Prerequisites

## Installation Steps

<Steps>
  <Step title="Clone the Repository">
    Clone the DriveLoop Mobile repository to your local machine:

    ```bash
    git clone <repository-url>
    cd driveloopMovil_test
    ```
  </Step>

  <Step title="Install Dependencies">
    Install all required dependencies using your preferred package manager:

    <CodeGroup>
      ```bash npm
      npm install
      ```

      ```bash yarn
      yarn install
      ```

      ```bash pnpm
      pnpm install
      ```
    </CodeGroup>

    <Info>
      This will install all dependencies including Expo Router, NativeWind (Tailwind CSS for React Native), React Navigation, and other required packages.
    </Info>
  </Step>

  <Step title="Verify Installation">
    Check that all dependencies are installed correctly:

    <CodeGroup>
      ```bash npm
      npm list --depth=0
      ```

      ```bash yarn
      yarn list --depth=0
      ```

      ```bash pnpm
      pnpm list --depth=0
      ```
    </CodeGroup>
  </Step>
</Steps>

## Setting Up Development Environment

### iOS Simulator (macOS only)

<Steps>
  <Step title="Install Xcode">
    Download and install Xcode from the Mac App Store.
  </Step>

  <Step title="Install Command Line Tools">
    ```bash
    xcode-select --install
    ```
  </Step>

  <Step title="Accept License">
    ```bash
    sudo xcodebuild -license accept
    ```
  </Step>
</Steps>

### Android Emulator

<Steps>
  <Step title="Install Android Studio">
    Download and install [Android Studio](https://developer.android.com/studio).
  </Step>

  <Step title="Install Android SDK">
    Open Android Studio and install:
    - Android SDK Platform 34 (or latest)
    - Android SDK Build-Tools
    - Android Emulator
  </Step>

  <Step title="Set Environment Variables">
    Add to your `.bashrc`, `.zshrc`, or equivalent:

    ```bash
    export ANDROID_HOME=$HOME/Library/Android/sdk
    export PATH=$PATH:$ANDROID_HOME/emulator
    export PATH=$PATH:$ANDROID_HOME/platform-tools
    ```

    <Note>
      On Windows, the Android SDK is typically located at `C:\Users\YourUsername\AppData\Local\Android\Sdk`
    </Note>
  </Step>

  <Step title="Create Virtual Device">
    Open Android Studio > Tools > Device Manager > Create Virtual Device
  </Step>
</Steps>

## Troubleshooting

### Port Already in Use

If port 8081 is already in use:

```bash
# Kill the process using port 8081
lsof -ti:8081 | xargs kill -9

# Or start on a different port
npx expo start --port 8082
```

### Module Not Found Errors

Clear cache and reinstall:

<CodeGroup>
  ```bash npm
  rm -rf node_modules package-lock.json
  npm install
  npx expo start --clear
  ```

  ```bash yarn
  rm -rf node_modules yarn.lock
  yarn install
  npx expo start --clear
  ```

  ```bash pnpm
  rm -rf node_modules pnpm-lock.yaml
  pnpm install
  npx expo start --clear
  ```
</CodeGroup>

### TypeScript Errors

Ensure TypeScript version matches the project:

```bash
npm install typescript@~5.9.2 --save-dev
```

### Font Loading Issues

If custom fonts don't load, clear the Expo cache:

```bash
npx expo start --clear
```
