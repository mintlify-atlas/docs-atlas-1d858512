## Overview

DriveLoop Mobile is a modern React Native mobile application built with Expo, featuring a complete authentication system and an intuitive dashboard interface. The app leverages the power of Expo Router for seamless navigation and NativeWind for beautiful, responsive styling.

## Technology Stack

DriveLoop Mobile is built on a modern tech stack:

- **React Native**: Cross-platform mobile development
- **Expo**: Streamlined development workflow
- **Expo Router**: File-based routing system
- **NativeWind**: Tailwind CSS for React Native
- **TypeScript**: Type-safe development
- **Lucide React Native**: Beautiful icon library
