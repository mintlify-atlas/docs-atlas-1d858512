## Overview

DriveLoop Mobile uses **NativeWind** to bring Tailwind CSS utility classes to React Native. The theme is defined in `tailwind.config.js` and includes custom colors and typography that reflect the DriveLoop brand.

## Color System

The app uses a carefully crafted color palette centered around a vibrant primary red.

### Primary Colors

<Tabs>
  <Tab title="Primary">
    ```js
    primary: {
      DEFAULT: "#C91843",
      dark: "#9B1B39",
      deep: "#870027",
    }
    ```
    
    The signature DriveLoop red used for buttons, links, and brand elements.
    
    <Note>
      **DEFAULT** (#C91843) is used for primary CTAs and interactive elements.
      **dark** (#9B1B39) provides hover states and emphasis.
      **deep** (#870027) offers maximum contrast for critical elements.
    </Note>
  </Tab>
  
  <Tab title="Secondary">
    ```js
    secondary: "#282828"
    ```
    
    Dark gray used for primary text and headings throughout the app.
  </Tab>
  
  <Tab title="Accent">
    ```js
    accent: "#111111"
    ```
    
    Near-black used for maximum contrast elements and icons.
  </Tab>
</Tabs>

### Tailwind Config

The complete theme configuration from `tailwind.config.js`:

```js tailwind.config.js
/** @type {import('tailwindcss').Config} */
module.exports = {
  content: [
    "./app/**/*.{js,jsx,ts,tsx}",
    "./components/**/*.{js,jsx,ts,tsx}"
  ],
  presets: [require("nativewind/preset")],
  theme: {
    extend: {
      colors: {
        primary: {
          DEFAULT: "#C91843",
          dark: "#9B1B39",
          deep: "#870027",
        },
        secondary: "#282828",
        accent: "#111111",
      },
      fontFamily: {
        'roboto-bold': ['Roboto-Bold', 'sans-serif'],
        'roboto-light': ['Roboto-Light', 'sans-serif'],
        'roboto-medium': ['Roboto-Medium', 'sans-serif'],
      }
    },
  },
  plugins: [],
}
```

## Typography System

DriveLoop Mobile uses **Roboto** as its primary typeface with three weights.

### Font Loading

Fonts are loaded in the root layout (`app/_layout.tsx`):

```tsx app/_layout.tsx
import { useFonts } from 'expo-font';
import { SplashScreen } from 'expo-router';

const RootLayout = () => {
    const [fontsLoaded, error] = useFonts({
        'Roboto-Bold': require('../assets/fonts/Roboto-Bold.ttf'),
        'Roboto-Light': require('../assets/fonts/Roboto-Light.ttf'),
        'Roboto-Medium': require('../assets/fonts/Roboto-Medium.ttf'),
    });

    useEffect(() => {
        if (error) throw error;
        if (fontsLoaded) SplashScreen.hideAsync();
    });

    if (!fontsLoaded && !error) return null;

    return <Slot />
}
```

<Warning>
  The splash screen is kept visible until fonts are fully loaded to prevent FOUT (Flash of Unstyled Text).
</Warning>

## Component Styling Patterns

### CustomButton Component

```tsx components/CustomButton.tsx
const CustomButton = ({ title, ...props }: CustomButtonProps) => {
    return (
        <TouchableOpacity
            className="bg-primary rounded-full py-4 items-center shadow-md active:opacity-80"
            {...props}
        >
            <Text className="text-white text-lg font-roboto-bold">{title}</Text>
        </TouchableOpacity>
    );
};
```

<Tabs>
  <Tab title="Colors">
    - `bg-primary` - Primary red background
    - `text-white` - White text for contrast
  </Tab>
  
  <Tab title="Shape">
    - `rounded-full` - Fully rounded pill shape
    - `py-4` - Vertical padding for touch target
  </Tab>
  
  <Tab title="Effects">
    - `shadow-md` - Medium shadow for depth
    - `active:opacity-80` - Press state feedback
  </Tab>
  
  <Tab title="Typography">
    - `text-lg` - Large text size
    - `font-roboto-bold` - Bold weight
  </Tab>
</Tabs>

### CustomInput Component

```tsx components/CustomInput.tsx
const CustomInput = ({ icon, ...props }: CustomInputProps) => {
    return (
        <View className="flex-row items-center border-b border-gray-300 py-2 mb-4">
            {icon && <View className="mr-3">{icon}</View>}
            <TextInput
                className="flex-1 text-base font-roboto-light text-secondary"
                placeholderTextColor="#9CA3AF"
                {...props}
            />
        </View>
    );
};
```

<Note>
  Input fields use a bottom border design (`border-b`) instead of full borders for a cleaner, more modern look.
</Note>

### MenuCard Component

```tsx components/MenuCard.tsx
const MenuCard = ({ title, Icon, onPress }: MenuCardProps) => {
    return (
        <TouchableOpacity
            onPress={onPress}
            className="flex-1 border border-primary/20 bg-white rounded-2xl p-6 items-center shadow-sm active:bg-gray-50 m-2"
        >
            <View className="mb-4">
                <Icon size={64} color="#C91843" strokeWidth={1.5} />
            </View>
            <Text className="text-secondary font-roboto-bold text-center text-lg">
                {title}
            </Text>
        </TouchableOpacity>
    );
};
```

<Tip>
  The `border-primary/20` class uses Tailwind's opacity modifier to create a subtle 20% opacity primary border.
</Tip>

## Customizing the Theme

### Adding Custom Colors

```js tailwind.config.js
theme: {
  extend: {
    colors: {
      primary: {
        DEFAULT: "#C91843",
        dark: "#9B1B39",
        deep: "#870027",
        light: "#E63959",  // Add lighter variant
      },
      success: "#10B981",   // Add success color
      warning: "#F59E0B",   // Add warning color
      error: "#EF4444",     // Add error color
    },
  },
},
```

### Adding Custom Font Sizes

```js tailwind.config.js
theme: {
  extend: {
    fontSize: {
      'xs': '12px',
      'sm': '14px',
      'base': '16px',
      'lg': '18px',
      'xl': '20px',
      '2xl': '24px',
      '3xl': '30px',
    },
  },
},
```

### Adding Custom Spacing

```js tailwind.config.js
theme: {
  extend: {
    spacing: {
      '18': '72px',
      '88': '352px',
      '128': '512px',
    },
  },
},
```

## Dark Mode Support

<Note>
  DriveLoop Mobile currently uses a light theme. To add dark mode support, extend the theme configuration:
</Note>

```js tailwind.config.js
theme: {
  extend: {
    colors: {
      dark: {
        bg: "#1F2937",
        text: "#F9FAFB",
        card: "#374151",
      },
    },
  },
},
```

Then use dark mode variants:

```tsx
<View className="bg-white dark:bg-dark-bg">
  <Text className="text-secondary dark:text-dark-text">
    Content
  </Text>
</View>
```

## Best Practices

<Steps>
  <Step title="Use theme colors consistently">
    Always use theme colors (`primary`, `secondary`, `accent`) instead of arbitrary hex values.
  </Step>
  
  <Step title="Maintain contrast ratios">
    Ensure text has sufficient contrast against backgrounds (WCAG AA minimum: 4.5:1).
  </Step>
  
  <Step title="Use opacity modifiers">
    Use Tailwind's opacity modifiers (`primary/20`) for subtle variations.
  </Step>
  
  <Step title="Test on multiple devices">
    Colors may appear different on iOS vs Android. Test on both platforms.
  </Step>
</Steps>
