## Overview

The `MenuCard` component is a stylized touchable card designed for dashboard menus and navigation grids. It displays a large icon with a title and provides visual feedback on press.

## Import

```typescript
import MenuCard from '../components/MenuCard';
```

## Props

<ParamField path="title" type="string" required>
  The text displayed below the icon
</ParamField>

<ParamField path="Icon" type="LucideIcon" required>
  A Lucide React Native icon component (not an instance, but the component itself)
</ParamField>

<ParamField path="onPress" type="() => void">
  Callback function triggered when the card is pressed
</ParamField>

## TypeScript Interface

```typescript
import { LucideIcon } from 'lucide-react-native';

interface MenuCardProps {
    title: string;
    Icon: LucideIcon;
    onPress?: () => void;
}
```

## Styling

The card includes these default styles:
- **Border**: Light primary color border with 20% opacity
- **Background**: White with gray background on active press
- **Shape**: Large rounded corners (`rounded-2xl`)
- **Padding**: 6 units all around, 2 units margin
- **Icon**: 64px size, primary color (`#C91843`), 1.5 stroke width
- **Text**: Secondary color, bold Roboto font, large size, centered
- **Shadow**: Small shadow for subtle depth
- **Layout**: Flex-1 for equal sizing in rows

## Component Source

Location: `~/workspace/source/components/MenuCard.tsx:11-25`

```typescript
const MenuCard = ({ title, Icon, onPress }: MenuCardProps) => {
    return (
        <TouchableOpacity
            onPress={onPress}
            className="flex-1 border border-primary/20 bg-white rounded-2xl p-6 items-center shadow-sm active:bg-gray-50 m-2"
        >
            <View className="mb-4">
                <Icon size={64} color="#C91843" strokeWidth={1.5} />
            </View>
            <Text className="text-secondary font-roboto-bold text-center text-lg">
                {title}
            </Text>
        </TouchableOpacity>
    );
};
```

<Note>
  The `Icon` prop expects a Lucide icon component (capital I), not an icon instance. Pass the component itself, and MenuCard will render it with the appropriate props.
</Note>

<Tip>
  Wrap MenuCards in a `flex-row` container to create horizontal layouts. The `flex-1` class ensures equal width distribution.
</Tip>

## Available Lucide Icons

Popular icons for menu cards:
- `User` - Account and profile
- `FileText` - Documents and files
- `Settings` - Configuration
- `Bell` - Notifications
- `Shield` - Security and privacy
- `HelpCircle` - Help and support
- `CreditCard` - Payments
- `Calendar` - Events and scheduling
- `MapPin` - Location services
- `Package` - Orders and deliveries
